# Jogo de Adivinhação em Java

Este é um simples jogo de adivinhação em Java, onde o computador gera um número aleatório entre 1 e 100, e o jogador tenta adivinhar esse número.

## Como Jogar

1. Compile o programa usando o comando `javac GuessNumber.java`.
2. Execute o programa usando o comando `java GuessNumber`.
3. O programa irá gerar um número aleatório entre 1 e 100.
4. Digite um palpite e pressione Enter.
5. O programa fornecerá dicas se o palpite é muito alto ou muito baixo.
6. Continue a adivinhar até acertar o número.

## Requisitos

- Java Development Kit (JDK) instalado.
